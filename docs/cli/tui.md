# `fluffbuzz tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
fluffbuzz tui
fluffbuzz tui --url ws://127.0.0.1:18789 --token <token>
fluffbuzz tui --session main --deliver
```
